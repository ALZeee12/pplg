Tentang RPL

Rekayasa Perangkat Lunak (RPL) adalah jurusan yang mempelajari tentang pengembangan perangkat lunak.

Mata Pelajaran

  • Pemrograman Dasar
  • Algoritma dan Struktur Data
  • Basis Data
  • Pengembangan Aplikasi Web

Karir

Lulusan jurusan RPL dapat bekerja sebagai:

  • Programmer
  • Analisis Sistem
  • Desainer Web
  • Pengembang Aplikasi
